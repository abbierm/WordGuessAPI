WordGuessAPI

Welcome! WordGuessAPI is a wordle clone you play via API calls. Ideally this was created to programmatically test wordle solvers without needing to manually input their answers into the terminal. WordGuessAPI is primarily written in Python with a bit of HTML and CSS for the website. I also used Flask as the web framework, SQLAlchemy/SQLite to manage the database, and Pytest for testing.

How To Play

Create a solver that can make the following POST requests:

  • https://www.wordguess_api.com/api/tokens
    • POST request for a 30 min api token
    • User must have an account and confirmed email address.
    • Requirements
      • Basic Auth header with username and password
    • Response will get returned as a JSON token in the form:
      • {"token": <"unique 30 min token goes here">}

  • http://www.wordguess_api.com/api/start
    • POST request that starts a new game.
    • Requirements
      • Auth token inside of the header in the following format:
        • {"Authorization": "Bearer: <"unique token">"}
      • Request body that contains the unique solver id
        • {"solver_id": <"unique solver_id found on solver's page>"}
    • Example using python requests: 
      • 
    import requests
    
    token_header = {"Authorization": "Bearer: my unique token"}

    request_body = {"solver_id": "my solver's unique api id"}

    r = requests.post(
        url="https://www.wordguess_api.com/api/start",
        headers=token_header,
        json=request_body
    )
    • Responses from the start and guess will be returned as a JSON in the following format: 
      • 
    {
        "game_token": 
        "solver_name": "The name you gave your solver",
        "status": "True if game is active else False",
        "guess_count": "0",
        "guesses": "Empty dictionary used to store guesses as received",
        "correct_word": "*****",
        "message": "Optional error message",
        "results": "'None' if game is still current else 'won' if the user won or 'lost'"
    }

  • http://www.wordguess_api.com/api/guess
    • POST request that makes a guess to game started via the start request above.
    • Requirements
      • Request body that contains the game token and a valid 5 letter word 
        
{
    "game_token": "game token received in starting payload",
    "guess": "5 letter guess"
}
    • Example using python requests: 
      • 
    import requests
    
    request_body = {"game_token": "the game's unique iD"}

    r = requests.post(
        url="https://www.wordguess_api.com/api/guess",
        json=request_body
    )
    • Response will be the same from making a start request. If the solver's guess was valid, the "guesses" value will be in the form: 
      • 
    {
        "1": {"guess": "user's 5 letter guess", "feedback": "5 letter feedback"}
    }
